# This is my final project for cs50x

## It is a game based on the old arcade game Donkey Kong.

I am using lua with LÖVE to make this game. I started teaching myself lua LÖVE on thursday Apr 2 2026
Before working on this project I haven't messed with lua much beside making small edits to neovim config.

### Running the project

to run this project you will need to install [LÖVE](https://www.love2d.org/)
then inside of the projects directory on your system you can run `love .` to play
